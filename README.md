# DedSecApp
DedSec Android App Using kivy

#### It's just a simple app with a Gif and Randomly generated texts

# Install
+ clone 
```bash
git clone https://github.com/Kourva/DedSecApp && cd DedSecApp
```
+ requirements
```bash
pip install kivy
```
+ run
```bash
python3 main.py
```
